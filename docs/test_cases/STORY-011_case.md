# Test Cases: STORY-011

> 注意：STORY-011 的 Spec 为空模板，需求从 PRD 推断。
> PRD 描述："基于配料偏好的商品推荐"

## TC-001: 推荐算法
**Given** 数据库中有多个商品，配料已识别
**When** `_get_recommendations()` 被调用
**Then** 返回配料评分最高的商品列表（评分 > 0），按评分降序排列

## TC-002: 首页推荐展示
**Given** 数据库中有历史搜索的商品数据
**When** 用户访问首页 GET /
**Then** 页面展示"推荐商品（配料评分最高）"区域，展示平台标识和评分

## TC-003: 推荐为空时不显示
**Given** 数据库中无商品数据
**When** 用户访问首页 GET /
**Then** 不显示推荐区域
