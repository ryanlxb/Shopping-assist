# Test Cases: STORY-007

## TC-001: 跨平台比价展示
**Given** 用户收藏了京东商品（¥12.90）和淘宝商品（¥9.90）
**When** 访问 `/favorites`
**Then** 页面展示价格范围 ¥9.90 ~ ¥12.90，价差 ¥3.00，且商品带平台标识

## TC-002: 按价格排序比价
**Given** 用户收藏了多个不同平台的商品
**When** 选择按价格升序排序 `/favorites?sort=price_asc`
**Then** 商品按价格从低到高排列

## TC-003: 配料评分对比
**Given** 用户收藏了多个商品
**When** 访问 `/favorites`
**Then** 每个商品展示配料评分和配料分类标签（颜色标记）
